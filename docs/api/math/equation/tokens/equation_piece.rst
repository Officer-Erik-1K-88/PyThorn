EquationPiece Base Type
=======================

.. py:class:: EquationPiece(value)
   :no-index:

Purpose
-------

``EquationPiece`` is the generic base for parsed equation tokens.

Members
-------

``value``
   Stored token value.

``parent``
   Parent equation piece if one exists.

``has_parent()``
   Return whether a parent is attached.
